Para instalar o APP Cidadão no Android:

1. Faça o Download do repositório https://github.com/maisa/iConv.git
2. Execute o pacote iconv.apk que está dentro do diretório iConv\App_Android\Instalador\

Para executar o APP Cidadão no Android:

1. Clique no ícone ICONV.
2. Clique no botão "Primeiro Acesso.
3. Na opção "Convênio Firmado com:" escolha a opção Prefeitura.
4. Escolha o Estado Rio Grande do Sul.
5. Escolha o Município Porto Alegre.
7. Clique em "OK".
8. Aparecerá a tela contendo a listagem de convênios referente a configuração efetuada nos passos anteriores. 
Para o protótipo, foi utilizado o convênio Número 774076, os dados foram obtidos do Governo Federal.
9. Clique no convênio Número 774076 para abrir a tela com os dados do convênio.
10. Para ver a prestação de contas deste convênio, clique no botão "Prestação de Contas Aqui".
11. Abrirá a tela contendo todos os pagamentos efetuados para este convênio.
12. Clique no botão "Voltar".
13. Para avaliar o convênio, clique no botão "Não".
14. Escolha o motivo pelo qual você não prova o convênio, é possível escolher mais de um motivo, e clique no botão "Confirma".

Observação: Foram incluídos alguns convênios para o Estado do Amazonas, porém não foi possível identificar pagamentos para os convênios escolhidos.
Os dados foram obtidos do Governo Federal.
